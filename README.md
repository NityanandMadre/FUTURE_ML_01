# Sales Forecasting Dashboard 

## Project Overview
This project builds an advanced sales forecasting dashboard using historical retail sales data. 
Time-series forecasting is performed using Facebook Prophet, and the results are visualized in Power BI.

## Tools & Technologies
- Python (Pandas, Prophet, Scikit-learn)
- Power BI
- Matplotlib
- Jupyter Notebook

## Dataset
Superstore retail sales dataset containing transactional order-level data.

## Methodology
1. Data cleaning and preprocessing
2. Time-series aggregation using Order Date
3. Forecasting future sales using Prophet
4. Model evaluation using MAPE
5. Dashboard creation in Power BI

## Key Features
- Actual vs Forecasted Sales Trend
- Actual vs Forecast Monthly sales Trend
- Business-focused visualization

## How to Run
1. Clone the repository
2. Install dependencies from requirements.txt
3. Run the Jupyter notebook
4. Refresh Power BI dashboard
